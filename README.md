# tree-sitter-zig

Zig grammar for [tree-sitter](https://github.com/tree-sitter/tree-sitter)

# Testing the parser
## Install dependency
```sh
npm install
```

## Test a zig file
```sh
./node_modules/tree-sitter-cli/tree-sitter parse <ZIG_FIlE>
```
